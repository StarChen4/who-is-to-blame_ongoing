### 1.3 分路与对线配置
**理解DOTA2的标准对线结构：**
- **优势路**（Safe Lane）：己方1号位+5号位 vs 对面3号位+4号位
- **中路**（Mid）：己方2号位 1v1 对面2号位
- **劣势路**（Off Lane）：己方3号位(+4号位游走) vs 对面1号位+5号位

需要明确：
- 每个人实际走的路
- 对线对手是谁（1v1还是2v2还是1v2）
- 是否换线/游走/三人路

## 🎮 第二步：英雄克制关系分析（辅助因素）

⚠️ **重要说明：克制关系只是众多考虑因素之一，不是核心决定因素！**

### 2.1 数据来源 ⭐
**唯一标准：https://dota2protracker.com/heroes**

访问步骤：
1. 进入 dota2protracker.com/heroes
2. 点击界面下方对应英雄头像
3. 查看 **"Countered By"**（被谁克制）
4. 查看 **"Good Against"**（克制谁）

**不要使用opendota的matchups数据！不要凭常识判断克制关系！**

### 2.2 对线克制快速判定

```
己方英雄 在protracker中：
- 在对手的"Good Against"列表 → 己方被克制
- 在对手的"Countered By"列表 → 己方克制对面

记录为：
Strong Disadvantage：被严重克制（列表前5名）
Slight Disadvantage：被轻微克制（列表6-10名）
Neutral：均势（不在列表中）
Slight Advantage：轻微克制（列表6-10名）
Strong Advantage：强力克制（列表前5名）
```

### 2.3 克制关系的合理使用

**对线期表现调整：**
```
克制调整系数：
- Strong Advantage（强力克制对面）：期望值+20%，表现差加畜
- Slight Advantage（轻微克制）：期望值+10%
- Neutral（均势）：标准评判
- Slight Disadvantage（轻微被克制）：容错+10%
- Strong Disadvantage（严重被克制）：容错+20%，表现差可理解
```

**重要：克制关系只影响期望值调整，不直接决定畜生认定！**

## 📋 第三步：BP阶段失误判定（辅助因素）

### 3.1 选人顺序理解

DOTA2的标准BP流程：
```
第1阶段：双方各Ban 2个 → 双方各选 2个（同步，互相看不到）
第2阶段：双方各Ban 1个 → 双方各选 2个（同步，互相看不到）
第3阶段：双方各Ban 1个 → 双方各选 1个（同步，互相看不到）

关键：同一阶段双方选择是同步盲选的！
```

### 3.2 BP失误判定规则

**BP失误的唯一判定条件（必须同时满足以下所有条件）：**

1. ✅ **位置要求**：选择的是核心位置（1、2、3号位）
2. ✅ **顺序要求**：不是己方最后一手选择（己方第5手）
3. ✅ **克制程度**：在dota2protracker.com的"Countered By"中，对面英雄的胜率达到43%以下
4. ✅ **实际结果**：对面确实选出了该克制英雄

**示例分析：**

❌ **不算BP失误的情况：**
1. 4、5号位辅助早选被克制 → 位置不符合条件
2. 己方最后一手（第5手）选核心被克制 → 已经是最后选了，无法避免
3. 胜率47%的克制关系 → 克制程度不够（>43%）
4. 对面没选克制英雄 → 虽然有风险，但未被punish

✅ **算BP失误的情况：**
```
己方第2手选择美杜莎(1号位)
→ protracker显示：敌法师 counter 美杜莎胜率41.4%
→ 对面第3手选择了敌法师
→ 满足所有4个条件，算BP失误！
```

### 3.3 BP失误权重

即使判定为BP失误，也仅作为**D级失误(4分)**，不能作为主要畜度来源。

## ⚖️ 第四步：位置职责标准

各位置核心职责与期望值（不区分英雄）：

### **1号位 (Carry)**
- ✅ 核心指标：后期伤害、GPM、团战生存
- 📊 期望值：GPM>500，伤害占比30-40%，团战不先死
- ⚠️ 责任：后期接管比赛，保持发育节奏
- ❌ 致命问题：吃满资源不出力、频繁送、后期疲软

### **2号位 (Mid)**
- ✅ 核心指标：对线表现、中期伤害、节奏控制
- 📊 期望值：15分钟等级领先、伤害占比25-35%、参团及时
- ⚠️ 责任：控符、中期节奏、游走支援
- ❌ 致命问题：对线崩、不游走、中期隐身

### **3号位 (Offlaner)**
- ✅ 核心指标：对线生存、参团率、开团/承伤
- 📊 期望值：对线不崩（经验差<20%）、参团率>65%、控制/承伤贡献
- ⚠️ 责任：抗压、开团、吸收火力
- ❌ 致命问题：对线喂太多、团战脱节、不先手

### **4号位 (Roaming Support)**
- ✅ 核心指标：游走效率、参团率、视野、节奏
- 📊 期望值：前期多次成功gank、插眼>10、参团>70%
- ⚠️ 责任：游走帮线、做视野、开雾抓人
- ❌ 致命问题：游走无效果、不做眼、抢核心钱

### **5号位 (Hard Support)**
- ✅ 核心指标：视野控制、保护核心、牺牲
- 📊 期望值：插眼>15、GPM<300、参团>60%、买辅助装
- ⚠️ 责任：全图视野、保1号位、买鸡买眼
- ❌ 致命问题：不买眼、抢补刀、存钱不花

## 🔥 第五步：失误分级与权重

**按严重程度排序（权重递减）：**

### **SSS级：战术/决策层面致命失误** (10分)
- 关键团战决策失误导致输掉比赛
  - 不该接团强接（-30k经济还主动开团）
  - 不该买活买活送（买活后立刻死）
  - 错误的带线决策（不该solo带线被抓）
- 阵容执行完全脱节
  - 推进阵容不推进、拖后期
  - 抓人阵容不开雾、打团
  - 后期阵容前期乱战

### **SS级：核心位资源严重浪费** (9分)
- 1/2号位拿大量资源却无贡献
  - GPM>450 但 伤害<20k（38分钟局）
  - 经济转化率全队最低（<0.5）
  - 正补>250 但伤害倒数
  - 团队资源占比>30% 但伤害占比<20%

### **S级：对线期严重失误** (8分)
- 均势/优势对线崩盘
  - 中路1v1被压（正补差>20）
  - 优势路2v2输给对面1个人
  - 前10分钟被单杀2+次
- 对线优势不转化
  - 前10分钟大优（正补领先30+）但后续疲软
  - 对线压制但不滚雪球

### **A级：中期节奏失误** (7分)
- 中期游走/支援严重失误
  - 2号位不游走（15-25分钟参团<3次）
  - 4号位不帮线（前15分钟gank成功<1次）
  - TP支援失误（队友被抓不TP/TP送人头）

### **B级：团战表现极差** (6分)
- 团战脱节/站位失误
  - 该开团不开（3号位躲后面）
  - 该放技能不放（控制技能捏手里）
  - 团战先死且无贡献（>8次团战先死）
- 参团率过低（<50%且无合理原因）

### **C级：经济/发育问题** (5分)
- 打钱效率极低
  - 核心位25分钟<15k净值
  - 长时间不打钱（5分钟补刀<20）
- 死亡经济损失过大
  - 带大额钱死亡（>1500金未花）

### **D级：辅助职责失职** (4分)
- 4/5号位不做视野（插眼<5）
- 4/5号位抢核心经济（正补>120）
- 4/5号位不买辅助装（无鸡/眼/烟/粉）
- 5号位不保护1号位
- **BP失误**（满足3.2的所有条件）

### **E级：频繁暴毙** (3分)
- 死亡过多（>12次）且多数可避免
- 暴毙率高（死亡时长>全队平均+3分钟）

### **F级：输出/贡献严重不足** (2分)
- 核心位伤害<全队15%
- 辅助位治疗/承伤/控制时间都很低
- KDA<0.8（5号位除外）

## 🎯 第六步：畜生认定算法

### 6.1 基础畜度计算
```
基础畜度 = Σ(各等级失误数 × 对应权重)

例如：
某玩家有：
- 1个SS级失误（资源浪费）→ 9分
- 2个S级失误（对线崩盘）→ 8×2=16分
- 1个B级失误（团战表现差）→ 6分
基础畜度 = 9+16+6 = 31分
```

### 6.2 调整系数应用
```
最终畜度 = 基础畜度 × 克制系数 × 位置系数 × 经济系数

克制系数（仅对对线期失误生效）：
- Strong Advantage（克制对面）：1.3（表现差加畜）
- Slight Advantage：1.1
- Neutral：1.0
- Slight Disadvantage：0.85（被克制，减免）
- Strong Disadvantage：0.7（被严重克制，大幅减免）

位置系数（平衡各位置畜度期望）：
- 1/2号位：1.0（标准，核心位责任大）
- 3号位：0.95（前排容错略高）
- 4号位：0.90（游走需要冒险）
- 5号位：0.85（硬辅容错最高）

经济背景系数：
- 大优势(>+20k)：1.0（表现差不可原谅）
- 小优势(+10~+20k)：1.0
- 均势(±10k)：1.0
- 小劣势(-10~-20k)：0.9（略微容错）
- 大劣势(-20~-35k)：0.8
- 巨劣势(<-35k)：0.7（全队劣势，个人难施展）
```

### 6.3 畜生数量控制机制 ⭐

**目标分布：**
- 1个畜生：70%的案例
- 2个畜生：27%的案例
- 3个畜生：3%的案例

**判定逻辑：**
```python
# 计算所有人的最终畜度
畜度排名 = sorted(五人畜度, reverse=True)

# 判定畜生数量
if 畜度排名[0] - 畜度排名[1] >= 15:
    # 第一名显著高于第二名 → 只有1个畜生
    畜生数量 = 1

elif 畜度排名[1] - 畜度排名[2] >= 12:
    # 前两名显著高于第三名 → 有2个畜生
    畜生数量 = 2

elif 畜度排名[2] - 畜度排名[3] >= 10:
    # 前三名显著高于其他人 → 有3个畜生（罕见）
    畜生数量 = 3

else:
    # 差距不明显，默认只认定1个最畜的
    畜生数量 = 1
```

### 6.4 位置平衡机制 ⭐

**目标：避免位置偏见，各位置都有合理的犯畜概率**

```python
# 历史统计（需要记录过往N场分析）
历史畜生位置分布 = {1号位: X%, 2号位: Y%, ...}

# 期望分布（可以略微倾向核心）
期望分布 = {
    1号位: 23%,  # 略高
    2号位: 23%,  # 略高
    3号位: 20%,  # 中等
    4号位: 17%,  # 略低
    5号位: 17%   # 略低
}

# 如果某位置历史占比过高，且当前畜度差距<10分，优先选其他位置
if 历史分布[某位置] > 期望分布[某位置] + 15%:
    if abs(畜度[该位置] - 畜度[次高]) < 10:
        # 优先认定其他位置为畜生
        选择次高畜度的不同位置
```

### 6.5 最终判定条件

**认定为畜生需同时满足：**
1. ✅ 最终畜度在全队排名前X（X=畜生数量）
2. ✅ 至少有2个B级(6分)以上的明确失误
3. ✅ 失误对比赛有实质影响（至少影响1波关键团战/节奏）
4. ✅ 考虑克制、位置、经济后，仍是主要问题来源
5. ✅ 符合数量控制机制（不超过3个）
6. ✅ 符合位置平衡机制（避免位置偏见）

**特殊优先级（多人接近时）：**
```
战术决策失误 > 资源浪费失误 > 对线失误 > 团战失误 > 基础操作失误
```

## 🔍 方法论核心原则总结

### ✅ 客观性原则
1. **数据驱动**：所有判断基于OpenDota数据
2. **标准统一**：克制关系仅以dota2protracker.com为准
3. **多维评估**：不依赖单一指标，综合评判

### ⚖️ 公平性原则
1. **克制关系是辅助因素**：不过分核心化，仅影响期望值调整
2. **BP失误严格判定**：
   - 必须同时满足4个条件才算BP失误
   - 辅助早选被克制不算失误
   - 即使算BP失误也仅4分（D级）
3. **位置平衡**：避免总是核心犯畜，各位置合理分布
4. **数量控制**：70%/27%/3%的1/2/3畜分布

### 🎯 科学性原则
1. **权重分级**：SSS(10) > SS(9) > S(8) > A(7) > B(6) > C(5) > D(4) > E(3) > F(2)
2. **系数调整**：克制×位置×经济 三重系数
3. **阈值判定**：畜度差距需>=10-15分才认定多畜
4. **历史追踪**：记录历史位置分布，动态平衡

### 🔄 动态平衡原则
1. **避免位置偏见**：如果某位置历史占比过高，适当平衡
2. **避免过度惩罚劣势局**：-35k以上劣势，经济系数0.7
3. **避免忽视优势局失误**：优势局表现差，不减免
4. **特殊情况优先**：战术失误 > 资源浪费 > 操作失误
